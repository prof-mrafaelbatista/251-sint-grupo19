Fundamentos da Programação em Python

Conceito: Permite que o programa tome decisões com base em condições usando if, elif e else.

Exemplo:

# Verifica se o usuário pode dirigir
idade = int(input("Informe sua idade: "))

if idade >= 18:
    print("Você pode tirar a carteira de motorista!")
else:
    print("Você ainda não pode tirar a carteira.")

Conceito: Executa um bloco de código repetidamente com for e while.

# Imprime os números pares de 1 a 10
print("Números pares de 1 a 10:")
for numero in range(1, 11):
    if numero % 2 == 0:
        print(numero)

# Repetição até o usuário digitar "sair"
comando = ""
while comando != "sair":
    comando = input("Digite um comando (ou 'sair' para encerrar): ")
    print(f"Você digitou: {comando}")

Conceito: Listas unidimensionais e bidimensionais usadas para armazenar dados.

# Lista de frutas
frutas = ["maçã", "banana", "laranja"]
print("Primeira fruta da lista:", frutas[0])

# Matriz de preços
tabela_precos = [
    [1, 2.50],
    [2, 3.00],
    [3, 4.25]
]
print("Preço do produto com ID 2:", tabela_precos[1][1])

Conceito: Funções retornam valores. Procedimentos apenas executam ações.

# Função com retorno
def aplicar_desconto(preco, percentual):
    return preco - (preco * percentual / 100)

# Procedimento (sem retorno)
def exibir_boas_vindas(nome):
    print(f"Bem-vindo(a), {nome}!")

preco_final = aplicar_desconto(100, 10)
print(f"Preço com desconto: R${preco_final:.2f}")
exibir_boas_vindas("Lucas")

Conceito: Permite capturar erros e evitar falhas no programa.

try:
    numero1 = float(input("Digite o numerador: "))
    numero2 = float(input("Digite o denominador: "))
    resultado = numero1 / numero2
    print(f"Resultado da divisão: {resultado:.2f}")
except ZeroDivisionError:
    print("Erro: não é possível dividir por zero.")
except ValueError:
    print("Erro: entrada inválida. Digite números válidos.")
finally:
    print("Operação finalizada.")
